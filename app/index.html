A Real-Time Comparison of Air Quality in Three Cities

About this graph: The graph above shows the last one hundred PM2.5 mass concentration values aggregated onto OpenAQ from public sources and pulled live from the OpenAQ API. The x-axis shows the local time for all station locations. Hover over datapoints to see the specific station locations within each city.

Want to access, copy and/or improve the code for this visualization? Do it here, it's all open.
