Beijing (Datapoints are updated approximately every hour and data represents hourly average):

  • Punjabi Bagh: Source | JSON

OpenAQ is an open, crowd-sourced platform that seeks to provide programmatic access to air quality (AQ) data from stations worldwide. Which stations OpenAQ reports data from depends on users like you. If you are interested in adding an air quality station to this platform, please contact us at info@openaq.org for more information.

All the data for this visualization is being pulled live from the OpenAQ platform which allows you to build whathever you want on top of it. It's all open-source so you can see the code the runs that platform and even make changes yourself at the GitHub repo.
